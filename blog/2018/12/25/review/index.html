Hilbert's spaces

Sea R∞\mathbb{R}^{\infty}R∞ que contiene todos los vectores con una sucesión infinita de componentes. Este espacio es actualmente muy grande cuando no existe control en el tamaño de las componentes vjv_{j}vj​ . Una idea mucho mejor es mantener la definición familiar de longitud, usando la suma de los cuadrados, e incluir solo aquellos vectores que tengan una longitud finita.

Longitud cuadrada∥v∥2=v12+v22+v32+⋯ . \textbf{Longitud cuadrada}\quad {\left|v\right|}^2= v_{1}^{2}+v_{2}^{2}+v_{3}^{2}+\cdots. Longitud cuadrada∥v∥2=v12​+v22​+v32​+⋯.

Esta serie infinita debe converger a una suma finita. Vectores con longitud finita pueden ser sumados, multiplicados por escalares, así ellos forman un espacio vectorial. En este espacio de Hilbert es la forma natural de crecer el número de dimensiones hasta el infinito, y al mismo tiempo mantener la geometría ordinaria de un espacio euclidiano. Las elipses serán elipsoides de dimensión infinita, y rectas perpendiculares serán exactamente como antes. Los vectores vvv y www son ortogonales cuando su producto interno es cero:

Suponga que f(x)=sin⁡xf\left(x\right)=\sin xf(x)=sinx sobre el intervalo 0≤x≤2π0\le x\le 2\pi0≤x≤2π . Esta suma es reemplazada, de modo natural e inevitable, por la integración:

Hat's functions

function grafica_hat(j, a, b, n, s)
% n segmentos
x = linspace(a, b, n + 1);
if j == 1
    xx = [x(1), x(2), x(n + 1)];
    yy = [1, 0, 0];
else
    if j == n
        xx = [x(1), x(n), x(n + 1)];
        yy = [0, 0, 1];
    else
        xx = [x(1), x(j - 1), x(j), x(j + 1), x(n + 1)];
        yy = [0, 0, 1, 0, 0];
    end
end
plot(xx, yy, s)
shg
end
