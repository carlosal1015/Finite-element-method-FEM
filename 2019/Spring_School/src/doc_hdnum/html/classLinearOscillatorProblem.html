Detailed Description

template<class T, class N = T>
class LinearOscillatorProblem< T, N >

Example class for a differential equation model.

The model is

u_0'(t) = -u_1(t), t>=t_0, u_0(t_0) = U_0 u_1'(t) = u_0(t), t>=t_0, u_1(t_0) = U_1

Template Parameters
Ta type representing time values
Na type representing components of states and f-values
